HTML Formatting Text Table

Tag Name Description Example
<b> Bold Bold Text, without any extra importance This text is bold
<strong> Strong Defines text with strong importance This text is strong
<i> Italic Tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc. This text is italic
<em> Emphasized A screen reader will pronounce the words in with an emphasis, using verbal stress. This text is emphasized
<mark> Mark Defines text that should be marked or highlighted This text is marked
<small> Small Defines smaller text This text is small
<del> Deleted Defines text that has been deleted from a document. Browsers will usually strike a line through deleted text This text is deleted
<ins> Inserted Defines a text that has been inserted into a document. Browsers will usually underline inserted text This text is inserted
<sub> Subscripted Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O This text is subscripted
<sup> Superscripted Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like WWW[1] This text is superscripted
